📌 Project Title

Personal Portfolio Website

📖 About the Project

This is my personal portfolio website developed using HTML, CSS, and JavaScript. It showcases my profile, skills, education, projects, and contact information in a clean and responsive design.

🎯 Objective

The main objective of this project is to create a responsive personal portfolio website to present my skills and work in a professional way.

✨ Features

- Responsive design (mobile, tablet, desktop)
- Home section with introduction
- About section
- Skills section
- Projects section
- Contact section
- Clean and simple UI

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

▶️ How to Run the Project

1. Download or clone the repository
2. Open the folder in VS Code
3. Open "index.html" in any browser
4. Or use Live Server extension in VS Code

📸 Output

The portfolio website displays a professional layout with sections like Home, About, Skills, Projects, and Contact. It is fully responsive and works on all devices.

📊 Result

The Personal Portfolio Website was successfully created using HTML, CSS, and JavaScript. It effectively showcases my skills, projects, and personal details in a structured format.

This project helped me improve my front-end development skills and understand how to build a responsive website.
